# 视频检测、去重与 IoU

IoU 是 Intersection over Union，中文常叫交并比，用来衡量两个边界框重叠程度。IoU = 交集面积 / 并集面积，取值 0 到 1。IoU 越高，说明预测框与真实框越接近。在目标检测中，IoU 阈值常用于判断预测是否命中，也用于 NMS 去掉重复框。

NMS 是 Non-Maximum Suppression，非极大值抑制。它会在同一目标附近保留置信度最高的框，并去掉与它 IoU 过高的重复框。提高 IoU 阈值会更宽松，可能保留更多重叠框；降低 IoU 阈值会更严格，可能减少重复框但也可能误删相邻车辆。

视频检测与单图检测不同。单图只统计当前画面中的目标；视频如果逐帧累加目标框，会把同一辆车在多帧中重复计算。视频预警更应关注采样帧数、每帧目标数趋势、跟踪 ID 去重后的车辆数、车辆类别分布、大型车辆比例和可见度变化。

摄像头实时检测通常每隔一定时间采样一帧送入后端 YOLO。为了降低卡顿，可以降低采样频率、缩小上传帧尺寸、避免前端重复绘制多个视频层，并在暂停或关闭摄像头时停止定时检测和 MediaStream 轨道。

当用户问“为什么视频检测车辆数很高”时，应解释：采样检测次数代表检测框出现次数，不等同于唯一车辆数；如果启用了跟踪去重，应优先查看去重车辆数；如果未启用或遮挡严重，重复统计会偏高。
