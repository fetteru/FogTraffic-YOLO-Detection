# 雨雾交通预警知识

本项目的预警目标是辅助雨雾、低光、摄像头画质下降等场景下的交通管理，不是单纯统计图片里有多少车。系统应综合可见度特征、车辆数量、大型车辆比例、平均置信度和视频采样稳定性来判断风险。

雨雾低能见度场景常见特征包括：画面整体亮度偏低、对比度下降、远处车道线和车辆轮廓模糊、目标框平均置信度下降、低置信度目标占比升高。晴天或可见度正常时，即使车辆数量较多，也不应直接判为高等级雨雾预警。

四级预警建议：

1. 1级，常规监测：可见度正常或仅有轻微画质波动，车辆较少或车流稳定。建议保持常规巡检。
2. 2级，关注提示：存在轻微雨雾、低光或画质下降，车辆较少或中等。建议关注车速和车距变化。
3. 3级，重点关注：存在明显雨雾或可见度较低，且车辆较多；或者可见度很低但车辆较少。建议提示驾驶员降速、保持车距，并保留关键视频片段复核。
4. 4级，应急处置：可见度很低且车辆很多，或大型车辆数量/占比明显偏高并叠加低能见度。建议交通管理人员立即关注现场，必要时限速、分流或发布预警。

大型车辆包括 truck、bus 等。大型车辆数量达到关注阈值时可以提升风险等级，因为其制动距离更长、遮挡范围更大，在雨雾条件下更容易放大追尾和连环事故风险。但大车数量不能脱离可见度单独触发最高等级，应与低能见度、车流密度共同判断。

摄像头视频需要注意去重：逐帧检测会把同一辆车重复统计多次。更适合作为预警依据的是去重后的车辆数、车流密度、采样帧趋势、大型车辆占比和平均置信度，而不是所有帧的目标框累加次数。
