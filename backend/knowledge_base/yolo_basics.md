# YOLO 基础知识

YOLO 是 You Only Look Once 的缩写，是一种单阶段目标检测算法。它把目标定位和类别预测放在同一次前向推理中完成，因此速度较快，适合实时检测、视频分析和边缘部署。

YOLO 模型通常输出边界框、类别和置信度。置信度表示模型认为该框包含目标且类别判断正确的程度。实际应用中可以通过调整 confidence threshold 控制检出数量，通过 IoU threshold 控制 NMS 去重强度。

YOLO11n 属于轻量级模型，参数量较小，适合在训练成本、推理速度和部署资源有限的场景中使用。对于遥感目标检测，可以通过迁移学习、数据增强和小目标优化提升效果。
