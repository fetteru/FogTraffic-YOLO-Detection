# YOLO-Highway-FogVehicleDetection
面向雨雾低光场景的高速轻量化车辆智能检测与拥堵预警系统（基于YOLO）

## 📖 项目简介
高速公路在大雾、降雨、夜间逆光、隧道暗光等低可视环境下，传统监控人工巡检效率低，通用目标检测模型易出现漏检、误检、推理卡顿等问题。
本项目基于轻量化YOLO算法，搭配**多阶段雨雾低光图像增强预处理**，实现高速路面多车型检测、多目标跟踪计数、车道车速估算、拥堵分级预警；支持RTSP摄像头视频流实时推理，可部署于边缘嵌入式设备，实现端边云一体化路况监测。

## ✨ 核心功能
- 🌫️ 雨雾/暗光/逆光图像增强：暗通道去雾、Retinex光照补偿、CLAHE对比度增强、双边滤波降噪
- 🚗 轻量化车辆目标检测：识别轿车、货车、大巴、摩托车、应急特种车辆
- 🔍 ByteTrack多目标跟踪：跨帧唯一ID绑定，车辆去重计数，过滤噪点假目标
- 🛣️ ROI车道自定义划分：限定检测区域，排除护栏与场外干扰
- 🚦 车速计算 + 车流量统计：单位时段车流、车型占比、瞬时密度统计
- ⚠️ 四级拥堵自动判别预警：畅通/缓行/拥堵/严重拥堵，本地日志+云端上报告警
- 📹 多源输入支持：本地视频、图片、USB摄像头、RTSP网络监控流
- ⚙️ 边缘端轻量化部署：支持ONNX导出、TensorRT/OpenVINO INT8量化加速

## 🎯 项目创新点
1. 构建串联式图像增强流水线，针对性修复雨雾退化图像特征丢失问题
2. 基于YOLO-N/Tiny轻量化主干+模型蒸馏+通道剪枝，大幅降低算力开销
3. 密度+车速双维度拥堵判定逻辑，比单一流量阈值更贴合高速真实路况
4. 端侧就地推理减少视频带宽传输压力，适配路侧工控机、RK3588、Jetson等硬件
